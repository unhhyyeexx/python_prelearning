### if문

#### if문

- 어떤 **조건을 만족**하는 경우, **명령문을 수행**하기 위해 사용
- true, false 둘중에 하나 반환
- 조건식에 : 붙이기
- 명령문 들여쓰기 꼭 통일

#### if~else문

- 어떤 **조건을 만족하는 경우의 명령문**과 **만족하지 않았을 경우의 명령문**을 상**호 배타적으로 수행**하고자 할 때 사용

#### if ~ elif ~ else 문

- 2개 이상의 다중 조건을 처리하고자 할 때 사용

- c에서 else if 와 같은 역할
